Java follows exactly the same principle. Before storing a value, Java needs to know what type of information is being stored.

What is a Data Type?

A Data Type tells Java what type of value a variable can store.

📘 Simple Definition

A Data Type specifies the type of data that can be stored inside a variable.

It also tells Java:

  • ✔ How much memory to allocate.
  • ✔ What type of values are allowed.
  • ✔ What operations can be performed.

💡 Why Do We Need Data Types?

Imagine every value is stored as plain text.

Age = "Twenty"

Price = "Five Hundred"

Marks = "Ninety"

Java cannot perform calculations on text values.

Now look at the correct way.

int age = 20;

double price = 500.50;

int marks = 90;

Now Java can calculate, compare, sort, and process the data correctly.

📚 Classification of Data Types

Java divides Data Types into two categories.

Primitive Data Types

Stores simple values directly.

  • byte
  • short
  • int
  • long
  • float
  • double
  • char
  • boolean

Non-Primitive Data Types

Stores references to objects.

  • String
  • Arrays
  • Classes
  • Objects
  • Interfaces

💡 In this lesson, we'll mainly focus on Primitive Data Types. You'll learn Non-Primitive Data Types in later modules.

📊 Java Primitive Data Types

Primitive Data Types are the basic building blocks of Java. They store simple values directly in memory and are faster than Non-Primitive Data Types.

Data Type Size Stores Example
byte 1 Byte Small Integer 100
short 2 Bytes Medium Integer 25000
int 4 Bytes Whole Numbers 500
long 8 Bytes Large Integer 1234567890L
float 4 Bytes Decimal Number 98.5f
double 8 Bytes Precise Decimal Number 98.5678
char 2 Bytes Single Character 'A'
boolean 1 Bit (Logical) true / false true

💻 Java Program Using Different Data Types


public class DataTypes{

public static void main(String args[]){

String name="Rahul";

int age=20;

double salary=35000.50;

char grade='A';

boolean passed=true;

System.out.println("Name : "+name);

System.out.println("Age : "+age);

System.out.println("Salary : "+salary);

System.out.println("Grade : "+grade);

System.out.println("Passed : "+passed);

}

}

🖥 Output

Name : Rahul

Age : 20

Salary : 35000.5

Grade : A

Passed : true

🔍 Line-by-Line Explanation

String name = "Rahul";

Stores the student's name as text.

int age = 20;

Stores age as a whole number.

double salary = 35000.50;

Stores salary with decimal precision.

char grade = 'A';

Stores one single character.

boolean passed = true;

Stores only true or false.

System.out.println()

Displays the stored values on the screen.

📜 Rules for Using Data Types

  • ✔ Choose the appropriate data type based on the value.
  • ✔ Store whole numbers using int whenever possible.
  • ✔ Store decimal values using double.
  • ✔ Store only one character using char.
  • ✔ Store only true or false using boolean.
  • ✔ Store text using String.

❌ Common Mistakes


int age = "20";

❌ Wrong because numbers should not be inside double quotes.


char grade = "A";

❌ Wrong because char stores only one character using single quotes.


char grade = 'A';

✔ Correct


boolean pass = Yes;

❌ Java accepts only true or false.


float price = 25.5;

❌ Missing f.


float price = 25.5f;

✔ Correct

💡 Best Practices

  • ✔ Use meaningful variable names.
  • ✔ Choose the smallest suitable data type whenever possible.
  • ✔ Use double for decimal values.
  • ✔ Use boolean for Yes/No conditions.
  • ✔ Use char only for one character.
  • ✔ Use String for text.
