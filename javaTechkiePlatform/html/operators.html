📖 What are Operators?

Operators are special symbols that tell Java to perform an operation on one or more values.

They allow Java to perform calculations, compare values, assign data, and make logical decisions.

Simple Definition

Operators are symbols used to perform different operations on variables and values.

🌍 Real-World Example

Imagine You Visit a Supermarket...

You purchase:

  • Milk – ₹60
  • Bread – ₹40
  • Chocolate – ₹100

The cashier calculates:

₹60 + ₹40 + ₹100 = ₹200

The + symbol performs addition. Java uses the same operator to perform mathematical calculations.

💼 Another Real-Time Example

Suppose your college allows only students with 75% attendance or above to write the semester examination.

Attendance >= 75

If the student's attendance is greater than or equal to 75, they are eligible for the exam.

Here, >= is a Relational Operator.

➕ Arithmetic Operators

Arithmetic operators are used to perform mathematical calculations.

Operator Name Example Result
+ Addition 10 + 5 15
- Subtraction 10 - 5 5
* Multiplication 10 * 5 50
/ Division 10 / 5 2
% Modulus 10 % 3 1

💻 Syntax

General syntax of arithmetic operators: 


result = operand1 operator operand2;

Example: 


sum = a + b;

💻 Java Example Program

The following program demonstrates all Arithmetic Operators. 


public class ArithmeticOperators {

    public static void main(String[] args) {

        int a = 20;
        int b = 5;

        System.out.println("Addition : " + (a+b));
        System.out.println("Subtraction : " + (a-b));
        System.out.println("Multiplication : " + (a*b));
        System.out.println("Division : " + (a/b));
        System.out.println("Modulus : " + (a%b));

    }

}

🖥 Output


Addition : 25
Subtraction : 15
Multiplication : 100
Division : 4
Modulus : 0

📖 Program Explanation

  • int a = 20;
    Declares an integer variable named a.

  • int b = 5;
    Declares another integer variable.

  • a + b
    Adds two numbers.

  • a - b
    Subtracts one number from another.

  • a * b
    Multiplies two numbers.

  • a / b
    Divides two numbers.

  • a % b
    Returns the remainder after division.

⚖ Relational Operators

Relational operators compare two values and always return either true or false.

Operator Meaning Example
> Greater Than 10 > 5
< Less Than 10 < 5
>= Greater Than or Equal 10 >= 10
<= Less Than or Equal 5 <= 10
== Equal To 10 == 10
!= Not Equal To 10 != 5

Example Program

int age = 18;

System.out.println(age >= 18);
System.out.println(age == 20);

Output

true
false

Real-Time Example

Online voting systems check whether a person's age is 18 or above before allowing them to vote.

🧠 Logical Operators

Logical operators combine two or more conditions.

Operator Meaning Example
&& Logical AND a>10 && b<20
|| Logical OR a>10 || b<20
! Logical NOT !true

Example Program

int age = 22;

boolean hasID = true;

System.out.println(age >= 18 && hasID);

Output

true

Real-Time Example

An ATM allows withdrawal only when:

  • ✔ Correct PIN entered
  • ✔ Enough account balance

Both conditions must be true.

📝 Assignment Operators

Assignment operators assign or update values stored inside variables.

Operator Example Equivalent To
= x = 10 Assign Value
+= x += 5 x = x + 5
-= x -= 2 x = x - 2
*= x *= 3 x = x * 3
/= x /= 2 x = x / 2

Example Program

int marks = 50;

marks += 10;

System.out.println(marks);

Output

60

Real-Time Example

When reward points are added after shopping, the system updates the total using +=.

🔄 Unary Operators

Unary operators work on only one variable.

Operator Meaning
++ Increment
-- Decrement

Example Program

int count = 10;

count++;

System.out.println(count);

Output

11

Real-Time Example

Every time a visitor opens a website, the visitor count increases by one.

❓ Ternary Operator

The Ternary Operator is a short form of the if-else statement.

Syntax

condition ? trueValue : falseValue;

Example Program

int marks = 85;

String result = (marks >= 50) ? "Pass" : "Fail";

System.out.println(result);

Output

Pass

Real-Time Example

An online examination system instantly displays Pass or Fail based on the student's marks.

🎯 Operator Precedence

Operator precedence determines the order in which Java evaluates an expression.

Priority Operators
1 () Parentheses
2 ++, --
3 *, /, %
4 +, -
5 > , < , >= , <=
6 == , !=
7 &&
8 ||
9 =

Example

int result = 10 + 5 * 2;

System.out.println(result);

Output

20

Multiplication has higher priority than addition.

❌ Common Mistakes

  • Using = instead of == for comparison.

  • Dividing a number by zero.

  • Ignoring operator precedence.

  • Using logical operators with invalid conditions.

  • Forgetting parentheses in complex expressions.

💡 Best Practices

  • ✔ Use meaningful variable names.
  • ✔ Keep expressions simple.
  • ✔ Use parentheses for better readability.
  • ✔ Test conditions before executing logic.
  • ✔ Avoid unnecessary calculations.
  • ✔ Comment complex expressions.
