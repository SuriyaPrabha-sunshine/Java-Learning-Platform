📘 What is a Variable?

A variable is a named memory location used to store data.

The value stored inside a variable can change during program execution.

Simple Definition

A variable is like a labeled container that stores information for later use.

💡 Why Do We Need Variables?

Imagine developing software without variables. Every value would have to be written repeatedly, making programs difficult to read and maintain.

Benefits of Variables

  • ✔ Store information efficiently
  • ✔ Reuse values anywhere in the program
  • ✔ Improve code readability
  • ✔ Simplify program maintenance
  • ✔ Support calculations and decision making
  • ✔ Essential for all real-world applications

💻 Variable Syntax

Every variable in Java follows a simple structure.

dataType variableName = value;

Example

int age = 20;

Explanation

Part Meaning
int Data Type
age Variable Name
= Assignment Operator
20 Value Stored
💡 Think of it as writing a label on a storage box.

📝 Variable Declaration

Declaration means creating a variable. At this stage, Java only reserves memory. No value is stored yet.

int marks;

Here, marks exists, but it doesn't contain any value.

🎯 Variable Initialization

Initialization means assigning a value to a variable.

marks = 95;

Now the variable stores the value 95.

🚀 Declaration and Initialization Together

Most Java programmers create and assign a value in one statement.

int marks = 95;

This is the most commonly used method.

🔍 Line-by-Line Explanation

String name = "Rahul";

Creates a variable called name and stores Rahul.

int age = 20;

Creates an integer variable named age.

double cgpa = 8.9;

Stores the student's CGPA.

System.out.println(name);

Displays the student's name.

System.out.println(age);

Displays the student's age.

System.out.println(cgpa);

Displays the student's CGPA.

📖 Variable Naming Rules

Rule Example
Must begin with a letter, _ or $ studentName ✔
Cannot begin with a number 1student ❌
No spaces allowed student name ❌
Cannot use Java keywords class ❌
Use meaningful names totalMarks ✔

❌ Common Mistakes

int 1age = 20;

❌ Variable names cannot start with numbers.

String name = Rahul;

❌ Strings must be inside quotation marks.

int marks = "90";

❌ Wrong data type assigned.

💡 Best Practices

  • ✔ Use meaningful variable names.
  • ✔ Follow camelCase naming.
  • ✔ Keep names short but descriptive.
  • ✔ Avoid unnecessary abbreviations.
  • ✔ Use lowercase for variable names.
